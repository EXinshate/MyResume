工作经验

后端开发实习生

二次方网络有限公司 | 2024年1月 - 2025年6月

项目名称：基于 Spring Cloud 的微服务商城系统

项目描述：开发了一个基于 Spring Cloud 的微服务商城系统，旨在为用户提供高效、可靠的在线购物体验。该系统采用微服务架构，主要包括以下核心组件：

  • Eureka Server：作为服务注册与发现中心，负责管理各个微服务的注册和发现，确保服务之间的动态调用和负载均衡。
  • Product Service：负责管理产品信息，包括产品的增删改查、分类管理、库存管理等功能。通过 Redis 缓存热点数据，将产品查询响应时间从 200ms 降低至 50ms。
  • Order Service：处理订单相关操作，包括订单创建、查询、更新和删除。使用消息队列（RabbitMQ）实现异步订单处理，提升系统吞吐量。
  • User Service：负责用户注册、登录、信息管理等功能。通过 JWT 实现用户身份验证和权限管理，确保用户数据的安全性。
  • API Gateway：作为系统的入口，负责请求路由、负载均衡、权限验证等功能。通过 Zuul 实现统一的 API 管理，简化了客户端与后端服务的交互。
  • 数据库：使用 MySQL 作为主要数据存储，采用分库分表策略以提高系统的可扩展性和性能。通过 Hibernate 实现 ORM，简化了数据库操作。

技术栈：Spring Cloud、Spring Boot、Spring Data JPA、Spring Security、Hibernate、MyBatis、MySQL、Redis、RabbitMQ、Git、Maven、Docker

项目成果：

  • 成功上线微服务商城系统，支持日均 10 万+用户访问，系统稳定性达到 99.9%。
  • 通过 Redis 缓存和消息队列优化，系统响应时间降低 60%，订单处理效率提升 50%。
  • 实现了高可用、可扩展的微服务架构，为后续功能扩展奠定了坚实基础。
